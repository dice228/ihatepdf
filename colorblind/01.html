🎲 3D LUT коррекция (.cube)
Применяется к изображению перед печатью в Photoshop, Lightroom, GIMP, darktable. Компенсирует ошибки принтера поточечно — точнее, чем 6 ползунков драйвера.
Как применить:
Photoshop → Image → Adjustments → Color Lookup → Load 3D LUT
Lightroom → Develop → Profile Browser → Import
GIMP → Colors → Color Mapping → Sample Colorize
darktable → 3D LUT module → Load .cube
⚠ Ограничения: LUT построен по 63 опорным точкам с интерполяцией (IDW). Точность зависит от качества скана. Для профессиональной печати используйте спектрофотометр + ICC.
